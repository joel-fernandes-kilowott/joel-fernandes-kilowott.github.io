my portfolio

Innovative Medium Of Learning Using AR APP : (Won 1st place for TECHNIX 2021 National Level Project Competition) We have built an AR learning android application (EDUCARE) using unity 3d software for engineering students which will help in better visualization of complex electronic circuits. It is a mobile application which uses the device camera to capture the circuit and then provides an augmented waveform simulation over that circuit. It also have additional features to calculate different factors of that circuit like Vdc, Idc, Irms etc. User is also able to listen to audio explanation of the scanned circuit.
